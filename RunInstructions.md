## How to run

To run the project:

1. Download the python script files either from the latest release or directly from the repository.
2. Upload files into a NeuroBlu Code Studio project
3. Select Main.py and click **run file**
4. The first time the project runs it will have to download the cohort dataset, subsequent runs will run based off the saved dataframe in the project
5. Finally the accuracy results will be returned in the console and graphs will be created to show most important features
